Форматы файлов.

.DOC или .dос (от англ. document)- расширение имени файла, иснолзуемое для файлов, предстаяляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файлов без формирования, однако позже стало использоваться для двоичных форматов с разметкой.
Документ MS Word

Portable Document Format (PDF)-межплатформенный формат электронных документов, разработанный фирмой Adobe Systems c использованием ряда позможностей языка Postscript.
Документ РDF

ZIP- популярный формат сжатия данных и архивации файлов. Файл этом формате обычно имеет расширение zip хранит в сжатом или несжатом виде одии или несколько файлов, которое можно из него извлечь путем распаковки с помощью специальной программы.
